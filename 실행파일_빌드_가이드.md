# 실행파일 빌드 가이드

## 사전 요구사항

- Python 3.11 이상 설치 (설치 시 "Add Python to PATH" 체크 필수)
- 인터넷 연결 (최초 패키지 설치 시 필요)

---

## 빌드 순서

### 1단계: 환경 설정 (최초 1회만)

`setup.bat`을 더블클릭하거나 터미널에서 실행한다.

```bat
setup.bat
```

수행 내용:
- Python 가상환경(`.venv`) 생성
- 필요한 패키지 설치 (`requirements.txt` 기준)
- Playwright Chromium 브라우저 설치

### 2단계: 실행파일 빌드

`build.bat`을 더블클릭하거나 터미널에서 실행한다.

```bat
build.bat
```

수행 내용:
- PyInstaller 설치
- `main.py`를 단일 실행파일로 패키징
- python-docx 템플릿, mistune, lxml 등 의존성 자동 포함

빌드 완료 후 결과물:

```
dist\md2docx.exe
```

---

## 배포

`dist\md2docx.exe` 파일 하나만 상대방에게 전달한다.

- Python 설치 불필요
- 더블클릭 시 서버 자동 시작 + 브라우저 자동 오픈

---

## Mermaid 다이어그램 관련

실행파일 안에 Chromium 브라우저는 포함되지 않는다. (포함 시 파일 크기 300MB 이상)

Mermaid 렌더링 우선순위:

| 순서 | 방법 | 조건 |
|------|------|------|
| 1 | Playwright (로컬 Chromium) | 개발 환경에서만 동작 |
| 2 | mermaid.ink API | 인터넷 연결 필요 |
| 3 | 텍스트 플레이스홀더 이미지 | 오프라인 완전 차단 시 |

**exe 배포 환경에서는 인터넷 연결이 있으면 mermaid.ink를 통해 정상 렌더링된다.**

## 자주 발생하는 오류

### `.venv not found` 오류
`setup.bat`을 먼저 실행하지 않은 경우.  
→ `setup.bat` 실행 후 다시 `build.bat` 실행

### `python`을 찾을 수 없음
Python이 설치되지 않았거나 PATH에 등록되지 않은 경우.  
→ [python.org](https://www.python.org/downloads/)에서 설치, 설치 시 "Add Python to PATH" 반드시 체크

### 빌드 후 실행 시 오류
바이러스 백신이 PyInstaller 실행파일을 차단하는 경우가 있음.  
→ 백신 예외 처리 후 재시도
